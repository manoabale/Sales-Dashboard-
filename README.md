# Sales-Dashboard- 📊 

A simple, responsive **Sales Dashboard** built with **HTML, CSS, and JavaScript**.  
It includes summary cards, a sidebar navigation, and interactive charts powered by **Chart.js**.

---

## 🚀 Features
- Sidebar navigation for quick access to sections
- Summary cards showing:
  - Total Sales
  - Orders
  - Customers
- Interactive bar chart for monthly sales
- Responsive design for desktop and mobile
- Easy to extend with more charts or tables

---

## 🛠️ Tech Stack
- **HTML5** – structure
- **CSS3** – styling
- **JavaScript (ES6)** – interactivity
- **Chart.js** – data visualization
